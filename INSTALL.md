# Zimport Installation Guide

## Download

Download the latest release from [Releases](https://github.com/hoosnick/zimport-app/releases).

## Installation

1. **Download** `zimport-X.X.X-windows.zip` from the latest release
2. **Extract** the archive to a folder of your choice
3. **Configure** - Edit `config.ini`
4. **Run** `zimport.exe`
